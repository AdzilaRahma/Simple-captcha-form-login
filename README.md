# Features
- Bootstrap 5.3.0 alpha3
- Bootstrap 5 Validation
- Verified by W3C
- Clean design
- Auto Disable/Enable Submit Button
- Easy to use
- Cross-browser compatibility (Chrome, Firefox, Opera, IE11, Safari)
- Fully responsive
- and more ...
